Get the current chrome profile
	In chrome, type chrome://version/ and get the Profile Path, removing "\Default".
	I.E.
		C:\Users\{user}\AppData\Local\Google\Chrome\User Data
	Paste that into windows explorer
	Copy the entire User Data folder to C:\Users\{name}\ChromeProfiles\User Data
Get the path to chromedriver in the project
	I.E. - C:\Users\{user}\source\repos\TubeBuddyScraper\packages\Selenium.WebDriver.ChromeDriver.77.0.3865.4000\driver\win32\chromedriver.exe
